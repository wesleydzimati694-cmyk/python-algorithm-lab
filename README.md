A dictionary is a data structure used to store information in key-value pairs.
Each key is unique and is used to access its corresponding value.
Dictionaries are useful because they allow fast lookup of data using keys instead of searching through a list.
Difference Between Linear Search and Binary Search
Linear Search

Linear search checks each element in a list one by one until it finds the target value or reaches the end.

How it works:

Start from the first element
Compare each element with the target
Stop when the target is found or the list ends

Key points:

Works on any list (sorted or unsorted)
Simple to implement
Slower for large datasets
Binary Search

Binary search is a more efficient method that works only on sorted lists. It repeatedly divides the search range in half.

How it works:

Start with the middle element
If the target is equal → found
If the target is smaller → search the left half
If the target is larger → search the right half
Repeat until found or no elements remain

Key points:

Requires a sorted list
Much faster than linear search for large datasets
More complex to implement
